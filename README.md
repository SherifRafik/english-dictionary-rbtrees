# english-dictionary-rbtrees

### Red black trees
  * Supported Methods:
      * Insertion
      * Deletion
      * Search
      * Get Minimum
      * Inorder Successor
      * Get black height
      * Get tree height

### English dictionary application
  * User can:
    * Load up a dictionary
    * Get dictionary size
    * Insert a new word
    * Look-up a word
    * Remove a word
    * Save the modified dictionary
